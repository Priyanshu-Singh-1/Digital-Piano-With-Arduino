1. Arduino UNO R3
2. Pushbuttons (7)
3. 10k ohm resistors (7)
4. Piezo/Buzzer (1)
5. Power Supply
6. Arduino IDE (download it from here - https://www.arduino.cc/en/software)

All the required folders and files are available in the master branch.
